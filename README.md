# Enhanced Wither Mod (ウィザー強化モッド)

本モッドは、Minecraftのボス「ウィザー」を極めて強力かつ凶悪に強化するモッドです。
強力なステータスに加え、弾幕シューティングのような多彩な攻撃パターンや巨大な爆発・デバフ攻撃を放つようになります。

## 機能・ステータス表

ウィザーの基本ステータスやシステム面の機能一覧です。ステータスは `EnhancedWitherConfig` にて変更可能です。

| 機能 / 項目 | 説明 | デフォルト値 (Config) | バニラ値 |
| :--- | :--- | :--- | :--- |
| **最大HP** (`maxHealth`) | ウィザーの最大体力です。 | 1000.0 | 300.0 |
| **アーマー値** (`armor`) | ウィザーの持つ防御力（アーマー値）です。 | 20.0 | 0.0 |
| **自動回復** (`regenPerSecond`) | 毎秒自動的に回復するHP量です（0で無効化）。 | 1.0 / 秒 | - |
| **ラグ対策** | プレイヤーから128ブロック以上離れたウィザースカルを毎秒消去し、パフォーマンス低下を防ぎます。 | - | - |
| **ドロップ強化** | 討伐時に豪華な報酬を追加でドロップします（ダイヤモンドブロック320個 ＋ ネザースター6個追加）。 | - | - |

---

## 攻撃パターン一覧表

ウィザーは2〜3秒（40〜60 ticks）のクールダウンを挟みながら、以下の7種類の攻撃からランダムに1つを選択して実行します。
また、全てのスカル攻撃において **5%の確率で青い頭蓋骨（無敵ブロック破壊可能）** が混ざります。

| 攻撃名 | タイプ | 継続時間 | 詳細 |
| :--- | :--- | :--- | :--- |
| **MACHINE_GUN<br>(マシンガン)** | 連続攻撃 | 10秒 | 対象に向かって高速でスカルを連射します（1秒間に10発）。 |
| **SPREAD_SHOT<br>(拡散ショット)** | 連続攻撃 | 10秒 | 対象の方向へ45度の円錐状にスカルをばらまきます（2ティック毎に15発発射）。 |
| **OMNI_DIRECTIONAL<br>(全方位弾幕)** | 連続攻撃 | 10秒 | フィボナッチ球体分布を用いて、全方位にスカルを大量発射します（2ティック毎に50発、秒間250発）。 |
| **EXPLOSIVE_SKULL<br>(超爆発スカル)** | 単発 | - | 着弾時に威力10の大爆発を引き起こす特殊なスカルを1発撃ちます。爆発範囲10ブロック内のエンティティに「ウィザー III」を10秒間付与します。 |
| **LIGHTNING<br>(落雷)** | 単発 | - | 対象の周囲（6×6ブロック）にランダムで5本の雷を落とします。落雷地点の3ブロック以内にいるエンティティに直接5.0のダメージを与えます。 |
| **DASH_ATTACK<br>(突進)** | 単発 | - | 対象に向かって超高速で突進します。進路（2.0ブロック範囲）にいるエンティティに20.0のダメージを与え、大きくノックバックさせます。 |
| **WITHER_POTION<br>(巨大ウィザーポーション)** | 単発 | - | 「ウィザー V」の効果を持つ特殊な残留ポーションを投擲します。着弾すると **半径100ブロック** に及ぶ巨大な毒霧（持続10秒）を発生させます。 |
