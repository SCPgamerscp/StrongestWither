# ウィザー強化MOD (Enhanced Wither Mod) 機能詳細

このMODは、マインクラフトのボスモンスター「ウィザー」を大幅に強化し、より絶望的で挑戦的な戦闘を提供するMODです。ステータスの強化だけでなく、弾幕シューティングゲームのような多彩で凶悪な攻撃パターンが追加されています。

---

## 1. 基本ステータスの強化
ウィザーの基礎能力が大幅に引き上げられています。

- **最大HPの増加**: バニラ(通常)の300から **1000** に大幅増加。
- **防御力の追加**: バニラの0から **アーマー値 20** が追加。物理的なダメージが通りにくくなっています。

---

## 2. 追加された特殊攻撃パターン（全7種類）
ウィザーは以下の7種類の強力な攻撃の中からランダムに1つを選び、2〜3秒（40〜60tick）の短いクールダウンを挟んで次々と繰り出してきます。
※すべてのウィザースカル（頭蓋骨）攻撃において、**5%の確率で青い頭蓋骨**（ブロックをより強力に破壊する頭蓋骨）が混ざります。

### ① マシンガン連射 (Machine Gun)
- **内容**: ターゲット（プレイヤーなど）に向かって、1秒間に10発の猛スピードでウィザースカルを連射します。
- **持続時間**: 10秒間連続で撃ち続けます。

### ② 前方拡散弾 (Spread Shot)
- **内容**: ターゲットの方向を中心とした45度の扇状（コーン状）に、無数のウィザースカルをバラ撒きます。
- **発射数**: 0.1秒(2tick)ごとに15発の頭蓋骨を放ちます。
- **持続時間**: 10秒間連続で撃ち続けます。回避が非常に困難な弾幕攻撃です。

### ③ 全方位弾幕 (Omni-directional Barrage)
- **内容**: ウィザーを中心に360度全方位へ向けてウィザースカルを発射します（フィボナッチ球による均等配置）。
- **発射数**: 0.1秒(2tick)ごとに50発（1秒間に250発）という驚異的な数の頭蓋骨を周囲にばら撒きます。
- **持続時間**: 10秒間連続で撃ち続けます。近づくことすら困難な制圧攻撃です。

### ④ 超爆発スカル (Explosive Skull)
- **内容**: 単発の特殊なウィザースカルを発射します。
- **爆発威力**: 着弾時に **爆発力10** （通常のクリーパーが3、TNTが4なので非常に広範囲）の巨大な爆発を起こします。
- **追加効果**: 爆発の範囲内（半径10ブロック以内）にいるプレイヤーやモブに、**「ウィザー III」の効果を10秒間**付与します。

### ⑤ 落雷攻撃 (Lightning Strike)
- **内容**: ターゲットの周囲にランダムで **5本の雷** を同時に落とします。
- **ダメージ**: 雷の直撃によるダメージや炎上に加え、落雷地点の半径3ブロック以内にいるエンティティに直接5ダメージ（ウィザーの攻撃扱い）を与えます。

### ⑥ 高速突進 (Dash Attack)
- **内容**: ターゲットに向かって超高速（速度3.0）で突進してきます。
- **ダメージと効果**: ウィザーの通り道（ウィザーから半径2ブロック以内）に巻き込まれた者に **20ダメージ** を与え、さらに上空へ大きく吹き飛ばす強いノックバックを与えます。

### ⑦ 巨大ウィザーポーション (Giant Wither Potion)
- **内容**: 放物線を描くように、特殊な残留ポーションを投げつけます。
- **効果範囲**: 着弾すると **半径100ブロック** という異常な広さのエフェクトクラウド（毒の霧）が一瞬で展開されます。
- **持続時間と効果**: 霧は10秒間その場に留まり、範囲内にいる者に**「ウィザー V（レベル5）」**の強烈なダメージを与え続けます。

---

## 3. パフォーマンス対策（ラグ軽減システム）
弾幕攻撃などによって一度に大量のウィザースカルが発射されるため、そのままではゲームの動作が極端に重くなってしまいます。このMODでは以下のラグ軽減対策を実装しています。

- **頭蓋骨の自動消去**: 1秒に1回、世界に存在するすべてのウィザースカルをチェックします。**一番近いプレイヤーから128ブロック以上離れてしまった頭蓋骨は自動的に消滅**する仕組みになっています。これにより、遠くの読み込まれていない場所へ飛んでいく無数の頭蓋骨が引き起こす処理落ちを未然に防ぎます。
